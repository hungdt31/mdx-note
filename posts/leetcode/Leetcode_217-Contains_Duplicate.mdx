## Description

Given an integer array `nums`, return `true` if any value appears at least twice in the array, and return `false` if every element is distinct.

## Example

### Example 1:

> **Input**: nums = [1,2,3,4]
<br/> **Output**: false

### Example 2:

> **Input**: nums = [1,2,3,1]
<br/> **Output**: true

### Example 3:

> **Input**: nums = [1,1,1,3,3,4,3,2,4,2]
<br/> **Output**: true

## Constraints
- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## Code

```c++
#include <iostream>
#include <vector>
#include <unordered_set>
using namespace std;
class Solution {
public:
  static bool containsDuplicate(vector<int>& nums){
    unordered_set <int> st;
    for (int num: nums){
      if (st.count(num) > 0) return true;
      st.insert(num);
    }
    return false;
  }
};
int main() {
  vector<int> nums;
  int n;
  cin >> n;
  for (int i = 0; i < n; i++){
      int num;
      cin >> num;
      nums.push_back(num);
  }
  cout << Solution::containsDuplicate(nums);
}
```
