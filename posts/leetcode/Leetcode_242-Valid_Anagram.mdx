Cho hai chuỗi s và t, trả về true nếu t là đảo chữ của s, và trả về false nếu ngược lại.

Đảo chữ là một từ hoặc cụm từ được hình thành bằng cách sắp xếp lại các chữ cái của một từ hoặc cụm từ khác, thường sử dụng tất cả các chữ cái gốc chính xác một lần.

## Example 

### Example 1

> Input: `s = "anagram", t = "nagaram"`
<br/> Output: `true`

### Example 2

> Input: `s = "rat", t = "car"`
<br/> Output: `false`

## Constraints:
- `1 <= s.length, t.length <= 5 * 104`
- s and t consist of lowercase English letters.

## Code

```c++
class Solution {
public:
    // sorting lettter
    bool isAnagramBySorting(string s, string t) {
        sort(s.begin(), s.end());
        sort(t.begin(), t.end());
        return s == t;
    }
    // using hash table
    bool isAnagramByHashing(string s, string t) {
        unordered_map<char, int> count;
        // Count the frequency of characters in string s
        for (auto x : s) {
            count[x]++;
        }
        // Decrement the frequency of characters in string t
        for (auto x : t) {
            count[x]--;
        }
        // Check if any character has non-zero frequency
        for (auto x : count) {
            if (x.second != 0) {
                return false;
            }
        }
        return true;
    }
};
```
